# 🎒 Chromebook 终极看海外网站保姆级教程（有手就会版）

> **一句话原理：**
> 学校限制了电脑，但没有限制 **GitHub**（写代码的作业网站）。我们利用 GitHub 免费借用一台海外的“云端电脑”，在里面套着看 Instagram、YouTube 等任何网站，学校的防火墙（Fortinet）绝对发现不了！

---

## 🛠️ 第一步：建立你的“云端电脑”

1. 在 Chromebook 上打开并登录 **`github.com`**（如果没有账号，用自己的邮箱免费注册一个，完全免费）。
2. 登录后，在浏览器地址栏输入这个网址进去：**`[github.com/codespaces](https://github.com/codespaces)`**。
3. 往下拉，找到一个写着 **"Blank"（空白模板）** 的方块，点击它上面的 **"Create codespace"** 按钮。
4. **耐心等待 1-2 分钟**，直到屏幕变成一个看起来很高级的代码界面，并且最下方出现一个黑色的正方形窗口（里面有一个闪烁的输入光标），这就说明云端电脑建好了。

---

## ⚡ 第二步：复制并运行“开机代码”

1. 完整复制下面这一行代码（一点都不要漏掉）：
```bash
docker run -d -p 8080:80 dorowu/ubuntu-desktop-lxde-vnc

```


2. 回到刚才那个黑色的窗口里，把代码**粘贴**进去，然后按下键盘上的 **回车键（Enter）**。
3. 此时黑框里会闪过几行英文字，不用管它，等它完全静止下来。

---

## 🌐 第三步：把画面“引”出来（最关键）

1. 仔细看屏幕的**右下角**，通常会弹出一个蓝色的提示框，写着 *"A service is running on port 8080..."*，直接点击它旁边的 **"Open in Browser"** 按钮。
2. **如果右下角没有弹出提示框，别慌，用手动方法：**
* 看屏幕左下角或者中间，有一个标签叫 **"Ports"（端口）**，点击它。
* 你会看到里面写着一个 `8080`。
* 点击 `8080` 旁边那个**像“小地球”一样的图标**（鼠标放上去会显示 *Open in Browser*）。



---

## 👑 第四步：在新世界快乐冲浪！

点击小地球后，你的浏览器会自动打开一个全新的网页标签页。

1. **见证奇迹：** 网页里会直接出现一个完整的 **Linux 电脑桌面**！
2. **上网：** 点击这个虚拟桌面左下角的蓝色菜单，打开里面自带的 **Firefox 或 Chrome 浏览器**。
3. **完美解决：** 在这个“画中画”的浏览器里输入 `instagram.com`。你想去什么网站就去什么网站，学校的防火墙拿你一点办法都没有！

---

## ⌨️ 附加：怎么在里面打中文？（必看）

因为这台云端电脑是纯英文的，直接按键盘打不出中文，也不能直接复制粘贴。

* **解决方法：** 看到运行 Linux 桌面的这个网页**左侧（或右侧）**，有一个隐藏的、半透明的**小箭头**吗？点击它会弹出侧边栏。
* 点击侧边栏里的 **Clipboard（剪贴板）** 图标。
* 把你在 Chromebook 上打好的中文，粘贴到这个黑色的文本框里，然后关闭侧边栏。
* 回到虚拟电脑的 Instagram 搜索框里，按下 `Ctrl + V`，中文就会神奇地出现了！

---

## 🛑 重点：下课怎么收尾？（不点会消耗额度！）

刷完之后，**千万不要直接关掉网页**！一定要让它“休眠”，这样下次来才能直接用，而且不消耗免费额度。

1. 放学前，回到 **`[github.com/codespaces](https://github.com/codespaces)`** 这个网页。
2. 找到你刚刚创建的这个空间，点击它右边的 **三个小点 `...**`。
3. 点击 **Stop Codespace（停止）**。

> **💡 这样做好处极大：** 云端电脑会自动关机。最厉害的是，你在里面登录的 Instagram 账号密码、历史记录都会被**完美保存**。下节课你来的时候，只需要重新点击它开机，一秒钟就能继续刷，再也不用输入任何代码了！
